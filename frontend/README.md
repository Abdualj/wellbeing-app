# Wellbeing App - Frontend

Simple React frontend showcasing the backend API features.

## Running

```bash
npm install
npm run dev
```

Frontend: http://localhost:5173  
Backend: http://localhost:3000

## Features

- Login with JWT authentication
- View user profile
- Create small groups
- Post to groups
- View group posts
